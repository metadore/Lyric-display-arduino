## 🧩 Hardware Used

| Component         | Description      |
| ----------------- | ---------------- |
| ⚡ Arduino Uno R4  | Main controller  |
| 📟 I2C LCD (16x2) | Displays lyrics  |
| 🌈 RGB LED Module | Lighting effects |
| 🔌 Jumper Wires   | Connections      |

---

## 🔌 Circuit Connections

### 📟 I2C LCD

| Pin | Connection |
| --- | ---------- |
| VCC | 5V         |
| GND | GND        |
| SDA | A4         |
| SCL | A5         |

---

### 🌈 RGB LED

| Color | Pin |
| ----- | --- |
| Red   | 6   |
| Green | 9   |
| Blue  | 3   |

> ⚠️ Use resistors if using raw LED

## 🛠️ Installation

```bash
1. Clone the repository
2. Open in Arduino IDE
3. Install library: LiquidCrystal_I2C
4. Upload code to Arduino
```

## 🎯 Usage

* Power the Arduino ⚡
* Play your song externally 🎧
* Watch lyrics + lights sync beautifully ✨

## 🎨 How It Works

* Uses **non-blocking timing (`millis()`)**
* Displays **fast, readable lyric chunks**
* RGB updates every ~200ms → simulates **music beats**
